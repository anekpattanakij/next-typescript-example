# TypeScript Next.js example

This is a really simple project that show the usage of Next.js with TypeScript.

## How to use it?


### Download next.js

```bash
npm install --save next -g
```
### Clone repository to your directory

### Install dependency and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
